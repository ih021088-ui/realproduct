# 로또 생성기 & AI 동물상 테스트

## 개요
로또 번호 생성, AI 동물상 테스트, 제휴 문의 기능을 제공하는 종합 엔터테인먼트 사이트입니다.

## 주요 기능
*   **AI 동물상 테스트 (New):** Teachable Machine 모델을 사용하여 강아지, 고양이, 고릴라 중 어떤 동물과 닮았는지 실시간으로 분석합니다.
*   **로또 생성기:** 6개의 행운 숫자를 생성합니다.
*   **제휴 문의 폼:** 파트너십을 위한 연락 창구입니다.
*   **댓글 시스템:** Disqus를 통한 사용자 소통 공간입니다.
*   **수익화:** Google AdSense를 통한 광고 수익 창출.

## 기술 스택
*   **AI:** TensorFlow.js, Teachable Machine Image Library
*   **Frontend:** HTML5, CSS3, JavaScript (Web Components)
*   **Backend:** Formspree (문의 폼)
*   **Ads:** Google AdSense

## Technology Stack

*   **HTML5 & CSS3 (Baseline):** Using Custom Properties, Flexbox, and modern selectors.
*   **JavaScript (ES Modules):** Organized into reusable Web Components.
*   **Formspree:** For serverless form handling.
*   **Google AdSense:** For monetization.
